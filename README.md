#  Cafe Sales Data Cleaning and Sales Analysis

## Project Overview

This project focuses on cleaning and analyzing a messy cafe sales dataset using Microsoft Excel. The raw dataset contained data quality issues such as missing values, inconsistent text formatting, duplicate records, and invalid entries. After cleaning the data, Pivot Tables and an interactive dashboard were created to analyze sales performance and provide actionable business insights.

## Project Objectives

- Clean and prepare the dataset for analysis.
- Improve data quality by correcting inconsistencies.
- Analyze sales performance across different categories.
- Create Pivot Tables to summarize key metrics.
- Design an interactive dashboard for business reporting.

## Dataset Description

The dataset contains transaction records from a café, including information such as:

- Order Date
- Product Name
- Product Category
- Quantity Sold
- Unit Price
- Total Sales
- Payment Method
- Customer Information
- Staff/Cashier
- Branch (where applicable)

## Data Cleaning Process

The following cleaning tasks were performed:

- Removed duplicate records.
- Removed blank rows.
- Handled missing values.
- Standardized text formatting.
- Corrected inconsistent spellings.
- Formatted dates and numeric values.
- Verified data consistency before analysis.

## Analysis Questions

The dashboard answers business questions such as:

- Which products generate the highest revenue?
- Which product categories perform best?
- What are the monthly sales trends?
- Which payment methods are most commonly used?
- Which products have the highest sales volume?

## Dashboard Features

- KPI Cards
- Sales Trend Analysis
- Revenue by Product
- Revenue by Category
- Payment Method Distribution
- Interactive Slicers
- Pivot Charts

## Key Insights

- Top-selling products contribute a significant share of total revenue.
- Some product categories consistently outperform others.
- Monthly sales trends reveal seasonal patterns.
- Customer payment preferences can guide future payment options.

## Recommendations

- Increase inventory for top-selling products.
- Promote lower-performing products through targeted offers.
- Monitor monthly sales trends for inventory planning.
- Continue maintaining clean and standardized sales records.
